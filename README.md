Sudoku generator, solver and interface to play games for both Node and Web.

## Installation

Using yarn:

```shell
$ yarn add sudokupower
```

Using npm:

```shell
$ npm i sudokupower
```

## Usage

```typescript
import { Game } from 'sudokupower`;
```
